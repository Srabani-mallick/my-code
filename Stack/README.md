Stack Implementation in Python

This  implements the **Stack Data Structure** using:

- Array (Static Implementation)
- Linked List (Dynamic Implementation)

## Operations

- Push
- Pop
- Display
- isEmpty
- isFull (Array Stack)

## Concepts Used

- LIFO (Last In First Out)
- Arrays
- Linked Lists
- Functions and Classes

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Push      | O(1) |
| Pop       | O(1) |
| Display   | O(n) |
